Bombs a specified user, or just yourself. This will also increment the bombed counter for the specified user.

If there isn't a mentioned user found in your message's contents, this will bomb yourself instead.

If there is a mention user found in your message's contents, this will bomb the pinged user instead.

If you want to view the leaderboard, you'll need to send this command: `/bomb null True`

## Arguments:
`/bomb (User<Member>) (Leaderboard<Boolean>)`

## Example Usage:
```
> /bomb
>> <@you> get bombed you bozo :joy:, user got bombed <@you's bombed counter> times

> /bomb @stranger
>> <@stranger> get bombed you bozo :joy:, user got bombed <@stranger's bombed counter> times

> /bomb null True
>> [leaderboard goes here]
```

-# There's a 50% chance every time to increase the amount of explosions you'll get in that one command.
